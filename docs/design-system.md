# Excedo Design System

Excedo uses a premium emerald, electric blue, and deep midnight palette instead of Jumia-style orange. It is designed for trust, speed, and marketplace depth.

## Tokens

- Midnight `#080B16`
- Navy `#10172A`
- Mist `#F5F7FB`
- Emerald `#10B981`
- Electric `#2563EB`
- Violet `#7C3AED`
- Gold `#F5B84B`
- Danger `#EF4444`

## Pages

Homepage, listing, product detail, cart, checkout, login/signup, customer dashboard, seller dashboard, and admin panel are responsive and use shared card, button, grid, and dashboard patterns.
