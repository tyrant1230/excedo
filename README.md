# Excedo

Premium ecommerce marketplace built with React, Tailwind CSS, Supabase, Stripe-ready checkout, and Vercel.

## Features

- Homepage with hero promo slider, categories, flash sales timer, product grid, newsletter, and footer.
- Product listing with filters and sorting controls.
- Product detail page with gallery, quantity selector, reviews, and related products.
- Cart and checkout with card payment and pay-on-delivery options.
- Login/signup screens wired for Supabase Auth.
- Customer dashboard, seller dashboard with analytics charts, and admin panel.
- Supabase migration for ecommerce tables and RLS policies.
- Vercel serverless Stripe checkout endpoint.
